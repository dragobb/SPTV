# StairPlay TV 📺✨

A premium, high-performance IPTV streaming application for Android, built with the latest modern technologies like **Jetpack Compose**, **Material 3**, and **Media3/ExoPlayer**.

## 🚀 Features

- **Premium UI/UX**: Ultra-modern dark mode design with Neon Purple accents and smooth animations.
- **Smart Country Detection**: Automatically detects your country to provide local channels, or manually switch between supported regions.
- **High-Performance Streaming**: Optimized player using ExoPlayer for low-latency and smooth playback.
- **Personalized Experience**: 
  - **Favorites**: Save your most-watched channels for quick access.
  - **Recently Watched**: Jump back into what you were viewing last.
- **Parental Lock**: Integrated Safe Mode to filter out NSFW/Adult content, protected by a secure PIN.
- **Flexible Playlists**: Add your own custom M3U/M3U8 playlists easily within the app.
- **Dual View Modes**: Switch between Grid and List views to suit your preference.

## 🛠 Tech Stack

- **Kotlin**: 100% modern programming language.
- **Jetpack Compose**: Declarative UI toolkit for smooth, efficient rendering.
- **Material 3**: The latest design standards from Google.
- **Media3 (ExoPlayer)**: Industry-leading media playback engine.
- **Room Database**: Local storage for channels, favorites, and search history.
- **Coroutines & Flow**: Asynchronous programming for high responsiveness.
- **Coil**: Image loading library for fast thumbnail and logo rendering.

## 🏗 How to Build

1. **Clone the repository**:
   ```bash
   git clone https://github.com/dragobb/StairPlay.git
   ```
2. **Open in Android Studio**:
   - Ensure you have **Android Studio Ladybug (2024.2.1)** or higher.
   - Recommended Java Version: **Java 17**.
3. **Build & Run**:
   - Sync project with Gradle files.
   - Click `Run 'app'` on your emulator or physical device.

## 📖 How to Use

1. **Getting Started**: On first launch, the app will attempt to detect your country and load a base playlist.
2. **Browsing**: Use the **Explore** tab to see all categories and featured content.
3. **Search**: Click the Search icon in the header to find specific channels instantly.
4. **Custom Playlists**: Go to **Settings -> Manage Playlists** to add your own IPTV sources.
5. **Safe Mode**: Toggle the NSFW filter in Settings to protect your viewing experience.
